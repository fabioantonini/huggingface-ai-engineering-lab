# Hugging Face AI Engineering Lab

Hands-on notebooks to learn how to build AI systems using
open-source models from Hugging Face.

Topics:

- Transformers
- NLP pipelines
- embeddings
- semantic search
- RAG
- chatbots
- multimodal AI
- voice AI

Run with Docker:

cd docker
docker compose up --build

Then open:

http://localhost:8888
